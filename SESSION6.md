String- sequence of characters. 

 

Lists - mutable (changeable), indexing/slicing, Duplicate elements, separated using commas inside square brackets, multiple data's can be stored, ordered (values are stored in index positions). 

Tuple – not mutable(no change), ordered, indexing/slicing, duplicate elements, normal brackets are used. 

Set – Mutable(changeable), not ordered, not indexing/slicing, no duplicate elements, Unique values are stored in sets, curly brackets are used. 

Dictionary - curly brackets are used, key values pair format
